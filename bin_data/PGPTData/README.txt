This folder contains the processed data for the PlayerGamesPlaytimeData class.
The files are pickled dictionaries with the keys 'lshensemble' and 'data'.
The 'lshensemble' key contains the MinHashLSHEnsemble object used to find similar users, and the 'data' key contains the processed data.

These are automatically generated when the class is first loaded. Delete to regenerate.

The filename contains the normalizer, approach to normalize, consideration for playtimes, output multiplier, threshold, number of permutations and number of partitions for the MinHash LSH Ensemble.
